# Game Landing Page - Forza Horizon 4

Este projeto é uma landing page do jogo Forza Horizon 4 desenvolvida com HTML e CSS.

## 📌 Sobre o projeto

O site apresenta informações sobre o jogo, galeria de imagens com lightbox, requisitos do sistema e opção de compra.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries (Responsivo)
* Lightbox com :target (CSS)

## 🎮 Funcionalidades

* Menu de navegação
* Hero section
* Informações do jogo
* Galeria com lightbox
* Navegação entre imagens
* Seção de compra
* Requisitos do sistema
* Call To Action
* Layout responsivo
* Footer

## 📁 Estrutura do projeto

```
game-landing-page/
│
├── index.html
├── style.css
└── img/
```

## 💻 Como executar

Basta abrir o arquivo `index.html` no navegador.
